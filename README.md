# 営業KPI・定性データ分析アプリ

Streamlit を使ったローテクWebアプリです。CSVデータをアップロードすることで、営業のKPIや定性データを可視化し、ベイジアンネットワークで簡易的な因果分析を行います。

---

## 🧰 使用技術

- Python
- Streamlit
- FirebaseやGitHubでの無料運用を想定（データ保存はローカル/CSVベース）

---

## 🚀 セットアップ手順（ローカル）

1. Python環境を準備
2. このリポジトリをクローン
3. 必要なパッケージをインストール:

```bash
pip install -r requirements.txt
